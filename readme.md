# Simple-Todo

## Commands:
**h**: To Show this Page <br>
**l**: Display all Todos.<br>
**a**: Add an new Todo.<br>
**e \<id>**: Edit the existing Todo.<br>
**r \<id>**: Remove a specific Todo based on the Id.<br>
**q**: Quit the Application<br>
